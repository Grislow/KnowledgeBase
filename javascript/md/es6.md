## New Numeric Literals
Binary literals start with `0b` or `0B` followed by a sequence of 1 or more binary digits.

Octal literals start with  `0o` or `0O` followed by a sequence of 1 or more octal digits.
``` javascript
0b10101                              // 21
0B1111111111                         // 1023
0o1234567                            // 342391
0O7777777                            // 2097151
```
